# Volume XXXII — Dell Technologies Certification Tracks

The Dell Technologies Proven Professional program, end to end: the
D-code framework, every current exam across nine portfolio areas, and
the platform knowledge behind them — anchored to this encyclopedia's
existing Dell volumes (XXII OpenManage, XXIII iDRAC, XXVI the R640
lab) and verified against Dell Learning's certification pages.

## Overview

Dell certifies by portfolio and function: Foundations, Operate,
Deploy, Design, Install, and Maintain credentials across servers,
storage, data protection, networking, HCI and cloud platforms, AI,
security, and client systems. This volume teaches each portfolio at
certification depth and closes with certification operations tuned to
the program's fast code churn.

## Certification alignment

This volume maps to the Dell Technologies Proven Professional
certifications, as recorded in
[CERTIFICATION_BLUEPRINTS.md](../../CERTIFICATION_BLUEPRINTS.md).
Chapter content describes exam scope and points to Dell's official
exam-description documents; it does not reproduce proprietary exam
questions or licensed courseware. **Verify the live code before every
booking** — this program rolls exam series (-23 → -01 → -A-00)
faster than any other on this shelf.

### The exams

All **77 current exams** below were verified against Dell Learning's
certification catalog on **22 July 2026**. Exams are delivered by
Pearson VUE; objectives and weights live in each exam's description
document on Dell Learning, which this volume treats as the authority.

## Study plans

**Foundations first** (two to three weeks each): D-ISM-FN-01 or the
portfolio Foundations nearest your role, on Chapter 01 plus the
portfolio chapter. **Operate tier** (four to six weeks): the platform
you run, its chapter's lab, and the exam description walked line by
line. **Deploy/Design tier** (six to eight weeks): add the adjacent
chapters this volume cross-references and produce the design
artifacts its labs demand. Re-verify every code the week you book.
