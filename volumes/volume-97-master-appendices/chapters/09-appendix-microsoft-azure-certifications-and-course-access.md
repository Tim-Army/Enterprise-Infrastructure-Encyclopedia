# Chapter 09: Appendix — Microsoft Azure Certifications and Course Access

Every current Microsoft Azure certification, organized by level, with its
exam code, any announced exam end date, and the official training that
prepares it. Every code and date below was harvested on **23 July 2026**
from Microsoft Learn — read from each certification's own page and the
public catalog API — the same harvest that anchors
[Volume XXXIII — Microsoft Azure Certification Tracks](../../volume-33-microsoft-azure-certifications/README.md),
whose nine chapters map this program to hands-on content.

**How access works at Microsoft.** Microsoft's model is the most generous
in this encyclopedia at the training layer. **[Microsoft Learn](https://learn.microsoft.com/en-us/training/)**
provides a **free training path for every certification below** — no
subscription, no partner, account optional for browsing. Many exams also
have a **free practice assessment**. Instructor-led training through
Microsoft Learning Partners is paid and optional. **Exams** are paid and
booked through your Microsoft Learn profile.

The distinctive part is **renewal**: role-based certifications are valid
for **one year** and are renewed by a **free, online, unproctored
assessment** available in the six months before expiry. Fundamentals
certifications **do not expire at all**.

> **Currency.** This program reorganizes faster than any other in this
> encyclopedia. Within one year: **AI-900 was renumbered to AI-901**
> (AI-900 retired 30 June 2026); **AI-102 and DP-100 retired**;
> **AZ-204 retires 31 July 2026 and AZ-500 retires 31 August 2026** —
> including their renewal assessments, so held credentials cannot be
> renewed past expiry; **three AI-centric associates arrived**, one on a
> brand-new `AB` code family; **DP-750** was added; **three specialties
> retired**; and **AZ-305 and AZ-400 were both updated**. Confirm any code
> and status on its Microsoft Learn page before registering. Volume
> XXXIII, Chapter 09 defines the recurring currency check.

## Free training and entry points

- **[Microsoft Learn training](https://learn.microsoft.com/en-us/training/)** —
  a free learning path for every certification below
- **Free practice assessments** — published for many exams, linked from
  each certification page
- **[Microsoft Learn Catalog API](https://learn.microsoft.com/api/catalog/)** —
  machine-readable certification and exam data, the primary source for the
  currency check
- **Free renewal assessments** — the annual, unproctored route to keeping
  a role-based certification current
- **Azure free account** — the practical lab substrate; pair it with a
  budget alert as Volume XXXIII, Chapter 01 sets up

## Reading the exam codes

The letter prefix identifies the subject family, not the level:

- **`AZ`** — core Azure platform (AZ-900, AZ-104, AZ-305, AZ-400)
- **`AI`** — artificial intelligence (AI-901, AI-103, AI-200)
- **`DP`** — data platform (DP-900, DP-300, DP-420, DP-750)
- **`SC`** — security, compliance, identity (SC-100, SC-200, SC-300)
- **`AB`** — AI agent building (AB-620), newly introduced

The number roughly tracks level — `-900` fundamentals, `-100`/`-200`
associate, `-300`/`-400` expert — but treat that as a pattern rather than
a rule: AI-103 and AI-200 are both associate.

## Notes on validity and cost

- **Fundamentals do not expire.** Role-based certifications (associate,
  expert, and specialty) are valid one year and renew free.
- **Renewal lapses silently.** The window is the six months before expiry;
  a missed window means re-earning by full exam.
- **Exam pricing** is shown at registration in your Microsoft Learn
  profile and varies by country; confirm it there rather than from any
  summary.
- **Free training covers every certification here**, which makes paid
  courses optional throughout and hard to justify at fundamentals level.
